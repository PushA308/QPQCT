Grading System

Evaluation criteria:


  • "This grading system is determined by considering Bloom's Taxanomy [BT], Course Coutcomes [CO],Question Types [T]"

  • The grading system adapt a six point scale which means awarding grades from A to F . This will evaluate the quality of question paper.

  • It is based on Two parameter:

    1. Absolute

    2. Percentile

  • Considered parameters for determining of good question paper :
    •                                            Grade
    1. BT = && T= && CO=       A
    2. BT = && T= && CO!=      B
    3. BT= && T!= && CO=       C
    4. BT= && T!= && CO!=      D
    5. BT!= && T= && CO=       E
    6. BT!= && T!= && CO!=     F
  • Grading Scale :

    • Grade Grade Point
    A 10
    B 9
    C 8
    D 7
